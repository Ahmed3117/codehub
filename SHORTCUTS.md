# Coder3 Keyboard Shortcuts

## Navigation & Switching
| Shortcut | Action |
|----------|--------|
| `Ctrl + P` | Open Command Palette (fuzzy search for sessions, apps, commands) |
| `Ctrl + 1..9` | Switch instantly to session 1 through 9 in the sidebar |
| `Ctrl + Tab` | Switch to the next session |
| `Ctrl + Shift + Tab` | Switch to the previous session |
| `Alt + 1..9` | Switch between app tabs (Editor, Postman, Chrome, etc.) within the active session |
| `Ctrl + B` | Toggle the sidebar visibility |

## Session Management
| Shortcut | Action |
|----------|--------|
| `Ctrl + N` | Create a new session |
| `Ctrl + Return` | Start the active session |
| `Ctrl + Shift + S` | Start the active session |
| `Ctrl + Shift + X` | Stop the active session |
| `Ctrl + Shift + R` | Force restart the active session (kills and relaunches all apps) |

## App Management
| Shortcut | Action |
|----------|--------|
| `Ctrl + W` | Close the currently active app tab (leaves the editor running) |

## Global Actions
| Shortcut | Action |
|----------|--------|
| `Ctrl + Shift + N` | Open General Notes & Plans |
| `Ctrl + Q` | Quit Coder3 (running sessions will be auto-restored on next launch) |
